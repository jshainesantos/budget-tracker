# Budget Tracker (React + DaisyUI)

Simple single-page budget tracker built with React, Vite, TailwindCSS and DaisyUI.

Features
- Log expenses and income quickly
- Basic categories (Food, Rent, Transport, Utilities, Entertainment, Other)
- Set budget limits per category
- Summary view with totals and balance
- Data persisted to `localStorage`

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

Files
- `src/` – React source (components, styles)
- `tailwind.config.cjs` – Tailwind + DaisyUI config
